# ENTERPOTATO

How Github works  
  -Create a new codespace using this repo as the template.  
  -When you want to change something go to the source control tab on the left  
  -3 dots right of changes  
    -Changes  
      -Stage all  
  -Down arrow next to commit  
    -Commit and push  
  -In the page that opens add a description or title. (DO NOT MAKE THIS A COMMENT).   
  -close the page
